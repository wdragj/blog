# мқёмҰқ л°©мӢқ


# API Key

м„ңл№„мҠӨл“Өмқҙ кұ°лҢҖн•ҙм§җм—җ л”°лқј кё°лҠҘл“Өмқ„ л¶„лҰ¬н•ҳкё° мӢңмһ‘н•ҳмҳҖлҠ”лҚ° мқҙлҘјмң„н•ҙ ModuleмқҙлӮҳ Applicationл“Өк°„мқҳ кіөмң мҷҖ лҸ…лҰҪм„ұмқ„ ліҙмһҘн•ҳкё° мң„н•ң кё°лҠҘл“Өмқҙ л“ұмһҘн•ҳкё° мӢңмһ‘н–ҲлӢӨ.

к·ё мӨ‘ м ңмқј лЁјм Җ л“ұмһҘн•ҳкі  к°ҖмһҘ л„җлҰ¬ ліҙнҺём ҒмңјлЎң м“°мқҙлҠ” кё°мҲ мқҙ API KeyмқҙлӢӨ.


# лҸҷмһ‘л°©мӢқ

  1. мӮ¬мҡ©мһҗлҠ” API KeyлҘј л°ңкёүл°ӣлҠ”лӢӨ. (л°ңкёү л°ӣлҠ” кіјм •мқҖ м„ңл№„мҠӨл“Өл§ҲлӢӨ лӢӨлҘҙлӢӨ. мҳҲлҘјл“Өм–ҙ кіөкіөкё°кҙҖ APIк°ҷмқҖ кІҪмҡ°м—җлҠ” мӢ мІӯм—җ н•„мҡ”н•ң м–‘мӢқмқ„ м ңм¶ңн•ҳл©ҙ кҙҖлҰ¬мһҗк°Җ нҷ•мқё нӣ„ KeyлҘј л°ңкёүн•ҙмӨҖлӢӨ.

  2. н•ҙлӢ№ APIлҘј мӮ¬мҡ©н•ҳкё° мң„н•ҙ KeyмҷҖ н•Ёк»ҳ мҡ”мІӯмқ„ ліҙлӮёлӢӨ.

  3. ApplicationмқҖ мҡ”мІӯмқҙ мҳӨл©ҙ KeyлҘј нҶөн•ҙ Userм •ліҙлҘј нҷ•мқён•ҳм—¬ лҲ„кө¬мқҳ Keyмқём§Җ к¶Ңн•ңмқҙ л¬ҙм—Үмқём§ҖлҘј нҷ•мқён•ңлӢӨ.

  4. н•ҙлӢ№ Keyмқҳ мқёмҰқкіј мқёк°Җм—җ л”°лқј лҚ°мқҙн„°лҘј мӮ¬мҡ©мһҗм—җкІҢ л°ҳнҷҳн•ңлӢӨ.


# л¬ём ңм җ

API KeyлҘј мӮ¬мҡ©мһҗм—җкІҢ м§Ғм ‘ л°ңкёүн•ҳкі  н•ҙлӢ№ KeyлҘј нҶөн•ҙ нҶөмӢ мқ„ н•ҳкё° л•Ңл¬ём—җ нҶөмӢ кө¬к°„мқҙ м•”нҳёнҷ”к°Җ мһҳ лҗҳм–ҙ мһҲлҚ”лқјлҸ„ Keyк°Җ мң м¶ңлҗң кІҪмҡ°м—җ лҢҖл№„н•ҳкё° нһҳл“ӨлӢӨ.

к·ёл Үкё°л•Ңл¬ём—җ мЈјкё°м ҒмңјлЎң KeyлҘј м—…лҚ°мқҙнҠёлҘј н•ҙм•јн•ҳкё° л•Ңл¬ём—җ лІҲкұ°лЎӯкі  мҳҲкё°м№ҳ лӘ»н•ң мғҒнҷ©(н•ңмӘҪл§Ң м—…лҚ°мқҙнҠёк°Җ мӢӨн–үлҗҳм–ҙ м„ңлЎң л§Өм№ҳк°Җ м•ҲлҗңлӢӨлҠ” л“ұ)мқҙ л°ңмғқн•  мҲҳ мһҲлӢӨ.

лҳҗн•ң, Keyн•ңк°Җм§ҖлЎң м •ліҙлҘј м ңм–ҙн•ҳкё° л•Ңл¬ём—җ ліҙм•Ҳл¬ём ңк°Җ л°ңмғқн•ҳкё° мү¬мҡҙнҺёмқҙлӢӨ.


# OAuth2

API Keyмқҳ лӢЁм җмқ„ л©”кҫёкё° мң„н•ҙ л“ұмһ‘н•ң л°©мӢқмқҙлӢӨ. лҢҖн‘ңм ҒмңјлЎң нҺҳмқҙмҠӨл¶Ғ, нҠёмң„н„° л“ұ SNS лЎңк·ёмқёкё°лҠҘм—җм„ң мүҪкІҢ ліј мҲҳ мһҲлӢӨ.

мҡ”мІӯн•ҳкі  мҡ”мІӯл°ӣлҠ” лӢЁмҲңн•ң л°©мӢқмқҙ м•„лӢҲлқј мқёмҰқн•ҳлҠ” л¶Җл¶„мқҙ м¶”к°Җлҗҳм–ҙ лҸ…лҰҪм ҒмңјлЎң м„ёл¶„нҷ”к°Җ мқҙлЈЁм–ҙмЎҢлӢӨ.


# лҸҷмһ‘л°©мӢқ

  1. мӮ¬мҡ©мһҗк°Җ Applicationмқҳ кё°лҠҘмқ„ мӮ¬мҡ©н•ҳкё° мң„н•ң мҡ”мІӯмқ„ ліҙлӮёлӢӨ. (лЎңк·ёмқё кё°лҠҘ, нҠ№м • м •ліҙ м—ҙлһҢ л“ұ лӢӨм–‘н•ң кіім—җм„ң м“°мқј мҲҳ мһҲлӢӨ. м—¬кё°м—җм„ңлҠ” лЎңк·ёмқёмңјлЎң нҶөмқјн•ҳм—¬ м„ӨлӘ…н•ҳкІ лӢӨ.)

  2. ApplicationмқҖ н•ҙлӢ№ мӮ¬мҡ©мһҗк°Җ лЎңк·ёмқёмқҙ лҗҳм–ҙ мһҲлҠ”м§ҖлҘј нҷ•мқён•ңлӢӨ. лЎңк·ёмқёмқҙ лҗҳм–ҙ мһҲм§Җ м•ҠлӢӨл©ҙ лӢӨмқҢ лӢЁкі„лЎң л„ҳм–ҙк°„лӢӨ.

  3. ApplicationмқҖ мӮ¬мҡ©мһҗк°Җ лЎңк·ёмқёлҗҳм–ҙ мһҲм§Җ м•Ҡмңјл©ҙ мӮ¬мҡ©мһҗлҘј мқёмҰқм„ңлІ„лЎң Redirectionн•ңлӢӨ.

  4. к°„м ‘м ҒмңјлЎң Authorize мҡ”мІӯмқ„ л°ӣмқҖ мқёмҰқм„ңлІ„лҠ” н•ҙлӢ№ мӮ¬мҡ©мһҗк°Җ нҡҢмӣҗмқём§Җ к·ёлҰ¬кі  мқёмҰқм„ңлІ„м—җ лЎңк·ёмқё лҗҳм–ҙмһҲлҠ”м§ҖлҘј нҷ•мқён•ңлӢӨ.

  5. мқёмҰқмқ„ кұ°міӨмңјл©ҙ мӮ¬мҡ©мһҗк°Җ мөңмҙҲмқҳ мҡ”мІӯм—җ лҢҖн•ң к¶Ңн•ңмқҙ мһҲлҠ”м§ҖлҘј нҷ•мқён•ңлӢӨ. мқҙлҹ¬н•ң кіјм •мқ„ Grantлқјкі  н•ҳлҠ”лҚ° лҢҖмІҙм ҒмңјлЎң мқёмҰқм„ңлІ„лҠ” мӮ¬мҡ©мһҗмқҳ мқҳм§ҖлҘј нҷ•мқён•ҳлҠ” GrantмІҳлҰ¬лҘј н•ҳкІҢ лҗҳкі , к°Ғ ApplicationмқҖ лӢӨмӢң к¶Ңн•ңмқ„ кҙҖлҰ¬ н•  мҲҳлҸ„ мһҲлӢӨ. мқҙ кіјм •м—җм„ң мӮ¬мҡ©мһҗмқҳ Grantк°Җ нҷ•мқёмқҙ лҗҳм§Җм•Ҡмңјл©ҙ лӢӨмӢң мӮ¬мҡ©мһҗм—җкІҢ Grantмҡ”мІӯмқ„ ліҙлӮёлӢӨ.

GrantлһҖ?

GrantлҠ” мқёк°ҖмҷҖлҠ” лӢӨлҘё к°ңл…җмқҙлӢӨ. мқёк°ҖлҠ” м„ңл№„мҠӨ м ңкіөмһҗ мһ…мһҘм—җм„ң мӮ¬мҡ©мһҗмқҳ к¶Ңн•ңмқ„ ліҙлҠ” кІғмқҙм§Җл§Ң, GrantлҠ” мӮ¬мҡ©мһҗк°Җ мһҗмӢ мқҳ мқёмҰқм •ліҙ(ліҙнҶө к°ңмқём •ліҙм—җ н•ҙлӢ№н•ҳлҠ” мқҙлҰ„, мқҙл©”мқј л“ұ)лҘј Applicationм—җ л„ҳкёём§Җ л§җм§Җ кІ°м •н•ҳлҠ” кіјм •мқҙлӢӨ.

  1. мӮ¬мҡ©мһҗк°Җ Grantмҡ”мІӯмқ„ л°ӣкІҢлҗҳл©ҙ мӮ¬мҡ©мһҗлҠ” н•ҙлӢ№ мқёмҰқм •ліҙм—җ лҢҖн•ң н—Ҳк°ҖлҘј лӮҙл ӨмӨҖлӢӨ. н•ҙлӢ№ мҡ”мІӯмқ„ нҶөн•ҙ лӢӨмӢң мқёмҰқм„ңлІ„м—җ мқёк°Җ мІҳлҰ¬лҘј мң„н•ҙ мҡ”мІӯмқ„ ліҙлӮҙкІҢ лҗңлӢӨ.

  2. мқёмҰқм„ңлІ„м—җм„ң мқёмҰқкіј мқёк°Җм—җ лҢҖн•ң кіјм •мқҙ лӘЁл‘җ мҷ„лЈҢлҗҳл©ҙ Applicationм—җкІҢ мқёк°ҖмҪ”л“ңлҘј м „лӢ¬н•ҙмӨҖлӢӨ. мқёмҰқм„ңлІ„лҠ” н•ҙлӢ№ мқёк°ҖмҪ”л“ңлҘј мһҗмӢ мқҳ м ҖмһҘмҶҢм—җ м ҖмһҘмқ„ н•ҙл‘”лӢӨ. н•ҙлӢ№ мҪ”л“ңлҠ” ліҙм•Ҳмқ„ мң„н•ҙ л§Өмҡ° м§§мқҖ кё°к°„лҸҷм•Ҳл§Ң мң нҡЁн•ҳлӢӨ.

  3. мқёк°Җ мҪ”л“ңлҠ” м§§мқҖ мӢңк°„ мң м§Җлҗҳкё° л–„л¬ём—җ мқҙм ң ApplicationмқҖ н•ҙлӢ№ мҪ”л“ңлҘј Request TokenмңјлЎң мӮ¬мҡ©н•ҳм—¬ мқёмҰқм„ңлІ„м—җ мҡ”мІӯмқ„ ліҙлӮҙкІҢлҗңлӢӨ.

  4. н•ҙлӢ№ Request Tokenмқ„ л°ӣмқҖ мқёмҰқм„ңлІ„лҠ” мһҗмӢ мқҳ м ҖмһҘмҶҢм—җ м ҖмһҘн•ң мҪ”л“ң(7лІҲ кіјм •)кіј мқјм№ҳн•ҳм§ҖлҘј нҷ•мқён•ҳкі  кёҙ мң нҡЁкё°к°„мқ„ к°Җм§Җкі  мӢӨм ң лҰ¬мҶҢмҠӨ м ‘к·јм—җ мӮ¬мҡ©н•ҳкІҢ лҗ  Access Tokenмқ„ Applicationм—җкІҢ м „лӢ¬н•ңлӢӨ.

  5. мқҙм ң ApplicationмқҖ Access Tokenмқ„ нҶөн•ҙ м—…л¬ҙлҘј мІҳлҰ¬н•  мҲҳ мһҲлӢӨ. н•ҙлӢ№ Access Tokenмқ„ нҶөн•ҙ лҰ¬мҶҢмҠӨ м„ңлІ„(мқёмҰқм„ңлІ„мҷҖлҠ” лӢӨлҰ„)м—җ мҡ”мІӯмқ„ н•ҳкІҢлҗңлӢӨ. н•ҳм§Җл§Ң мқҙ кіјм •м—җм„ңлҸ„ лҰ¬мҶҢмҠӨ м„ңлІ„лҠ” л°”лЎң лҚ°мқҙн„°лҘј м „лӢ¬н•ҳлҠ” кІғмқҙ м•„лӢҢ мқёмҰқм„ңлІ„м—җ м—°кІ°н•ҳм—¬ н•ҙлӢ№ нҶ нҒ°мқҙ мң нҡЁн•ңм§Җ нҷ•мқёмқ„ кұ°м№ҳкІҢлҗңлӢӨ. н•ҙлӢ№ нҶ нҒ°мқҙ мң нҡЁн•ҳлӢӨл©ҙ мӮ¬мҡ©мһҗлҠ” л“ңл””м–ҙ мҡ”мІӯн•ң м •ліҙлҘј л°ӣмқ„ мҲҳ мһҲлӢӨ.



# л¬ём ңм җ

кё°мЎҙ API Keyл°©мӢқм—җ л№„н•ҙ мўҖ лҚ” ліөмһЎн•ң кө¬мЎ°лҘј к°Җм§„лӢӨ. л¬јлЎ  л§ҺмқҖ л¶Җл¶„мқҙ к°ңм„ лҗҳм—ҲлӢӨ.

н•ҳм§Җл§Ң нҶөмӢ м—җ мӮ¬мҡ©н•ҳлҠ” TokenмқҖ л¬ҙмқҳлҜён•ң л¬ёмһҗм—ҙмқ„ к°Җм§Җкі  кё°ліём ҒмңјлЎң м •н•ҙм§„ к·ңм№ҷм—Ҷмқҙ л°ңн–үлҗҳкё° л•Ңл¬ём—җ мҰқлӘ…нҷ•мқё н•„мҡ”н•ҳлӢӨ.

к·ёл Үкё°м—җ мқёмҰқм„ңлІ„м—җ м–ҙл–Ө мӢқмқҙл“  DBMS м ‘к·јмқҙл“  лӢӨлҘё APIлҘј нҷңмҡ©н•ҳм—¬ м ‘к·јн•ҳлҠ” л“ұмқҳ мң нҡЁм„ұ нҷ•мқё мһ‘м—…мқҙ н•„мҡ”н•ҳлӢӨлҠ” кіөмҰқ м—¬л¶Җ л¬ём ңк°Җ мһҲлӢӨ. мқҙлҹ¬н•ң кіөмҰқм—¬л¶Җ л¬ём ңлҝҗл§Ң м•„лӢҲлқј мң нҡЁкё°к°„ л¬ём ңлҸ„ мһҲлӢӨ.


# JWT

JWTлҠ” JSON Web Tokenмқҳ мӨ„мһ„л§җлЎң мқёмҰқ нқҗлҰ„мқҳ к·ңм•Ҫмқҙ м•„лӢҢ Token мһ‘м„ұм—җ лҢҖн•ң к·ңм•ҪмқҙлӢӨ. кё°ліём Ғмқё Access TokenмқҖ мқҳлҜёк°Җ м—ҶлҠ” л¬ёмһҗм—ҙлЎң мқҙлЈЁм–ҙм ё мһҲм–ҙ Tokenмқҳ м§„мң„лӮҳ мң нҡЁм„ұмқ„ л§ӨлІҲ нҷ•мқён•ҙм•ј н•ҳлҠ” кІғмһ„м—җ л°ҳн•ҳм—¬, JWTлҠ” мқёмҰқм—¬л¶Җ нҷ•мқёмқ„ мң„н•ң к°’, мң нҡЁм„ұ кІҖмҰқмқ„ мң„н•ң к°’ к·ёлҰ¬кі  мқёмҰқ м •ліҙ мһҗмІҙлҘј лӢҙкі  мһҲкё° л•Ңл¬ём—җ мқёмҰқм„ңлІ„м—җ л¬»м§Җ м•Ҡкі лҸ„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

нҶ нҒ°м—җ лҢҖн•ң мһҗм„ён•ң лӮҙмҡ©кіј мқёмҰқл°©мӢқмқҖ JWTл¬ём„ң (opens new window)лҘј м°ёмЎ°н•ҳмһҗ.


# л¬ём ңм җ

м„ңлІ„м—җ м§Ғм ‘ м—°кІ°н•ҳм—¬ мқёмҰқмқ„ н•ҷмқён•ҳм§Җ м•Ҡм•„лҸ„ лҗҳкё° л•Ңл¬ём—җ мғқкё°лҠ” мһҘм җл“Өмқҙ л§ҺлӢӨ. н•ҳм§Җл§Ң нҶ нҒ° мһҗмІҙк°Җ мқёмҰқ м •ліҙлҘј к°Җм§Җкі  мһҲкё°л•Ңл¬ём—җ лҜјк°җн•ң м •ліҙлҠ” мқёмҰқм„ңлІ„м—җ лӢӨмӢң м ‘мҶҚн•ҳлҠ” кіјм •мқҙ н•„мҡ”н•ҳлӢӨ.
